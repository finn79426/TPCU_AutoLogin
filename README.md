## 開始
你至少需要：
* Clone 或是 dowload **TPCU_AutoLogin**
* Python3.5
* Chrome安裝版 (請先事前安裝完成)

## 代碼說明
config.txt 設定檔，需在裡面設定帳號、密碼等<br>
login.py 主程式，內有處理驗證碼，尋找元素的code

## 如何安裝？
首先，請先到 www.python.org 下載Python<br>
請至少選擇3.5以上的版本，並請在安裝過程勾選安裝pip
